# **State Machine Editor**

ブラウザ上で動作する、直感的なステートマシン（状態遷移図）エディタです。
作成した図からC言語のスケルトンコードをリアルタイムに生成し、組み込みプロジェクト等に活用できます。

## **🚀 主な機能**

### **1\. ダイアグラム編集**

* **ノード操作**: 通常状態（State）、初期状態（Initial）、条件分岐（Choice）の追加・配置。  
* **遷移線（Transition）**: ノード間のドラッグ＆ドロップで接続。中継点（Waypoint）の追加による線の折り曲げに対応。  
* **複数選択**: キャンバスのドラッグによる範囲選択、または Shift \+ クリック での個別選択が可能。

### **2\. ファイル操作（モダンAPI対応）**

* **上書き保存**: 一度保存・読込をしたファイルに対して、ダイアログなしで直接上書き保存が可能。  
* **名前を付けて保存**: 常に新しいファイルとしてデバイスに保存できます。  
* **プロジェクト読込**: 保存済みの .json ファイルを読み込み、編集状態を完全に復元します。

### **3\. ショートカットキー**

効率的な編集のために、以下のショートカットに対応しています。

| キー組合せ | 動作内容 |
| :---- | :---- |
| **Ctrl \+ S** | 上書き保存 (Save) |
| **Ctrl \+ O** | ファイルを開く (Open) |
| **Ctrl \+ Z** | 元に戻す (Undo) |
| **Ctrl \+ Y / Shift \+ Z** | やり直し (Redo) |
| **Ctrl \+ C** | コピー (Copy) |
| **Ctrl \+ X** | 切り取り (Cut) |
| **Ctrl \+ V** | 貼り付け (Paste) |
| **Ctrl \+ A** | 全ての要素を選択 (Select All) |
| **Delete / Backspace** | 選択中の要素を削除 (Delete) |
| **Escape** | 選択解除 / メニューを閉じる / 遷移作成のキャンセル |

### **4\. その他の機能**

* **Internal Actions**: 各状態に Entry, Do, Exit のアクションコードを記述可能。変更があった場合のみ履歴に保存されます。  
* **コード生成**: 状態遷移に基づいたC言語のスイッチケース・構造体コードを Code ビューで即座に確認・コピー可能。
